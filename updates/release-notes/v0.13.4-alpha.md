## v0.13.4-ALPHA 设置与更新链路打磨

这是 Alpha 测试版本，重点继续收紧桌面端设置、MCP 包管理和软件更新体验。

### 更新内容

* 设置偏好改为优先写入桌面端本地 `settings.json`，本地服务不可用时也能保留修改。
* MCP 包管理区优化运行时版本选择、安装、卸载和自适应布局。
* DEV 调试入口改为隐藏机制：1 秒内连续点击 MCP 更新图标 5 次后解锁，且不跨启动保存。
* 开发者模式仅在本次会话生效，`v99.99` 测试更新只在 DEV 开启后出现。
* 软件更新列表补充发布时间读取，避免历史版本显示未知时间。
* 侧栏禁用页面折叠增加动画，并修正箭头方向。
* 桌面端增加设置偏好文件读写命令，减少设置页保存失败带来的误导。

### 已知说明

当前仍是 Alpha 版本。首次启动、MCP 安装、项目识别和更新覆盖安装链路还在继续打磨；如果遇到卡住、闪退、更新失败或提示看不懂，可以导出诊断包后反馈。
